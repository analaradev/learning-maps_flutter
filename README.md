## Contexto

Repositorio de aprendizaje para explorar mapas, ubicaciones y componentes relacionados con geolocalizacion en aplicaciones moviles.

## Proposito

- Visualizar mapas dentro de una app.
- Practicar componentes de ubicacion.
- Explorar funcionalidades basadas en posicion.
- Reforzar desarrollo movil con Flutter.
